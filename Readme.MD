# Pre-requisitos
1. Tener instalado MongoDB en el equipo junto a MongoCompass (O RoboMongo).
2. Los binarios de mongo deben estar registrados en el PATH para poder correr el comando "mongod".
3. Tener Node.js (al menos la version v20.18.2 o superior).
3. Tener instalado el CLI de Angular en su version 18+.

## Instrucciones para correr el proyecto
1. Se debe iniciar mondoDB para poder registrar los datos.
```
    C:\> mongod
```
